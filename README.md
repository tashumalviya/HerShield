# HerShield

HerShield is a Women Safety Web Application designed to provide quick access to emergency assistance through features like SOS alerts, current location, emergency contacts, and an AI safety chatbot.

## Features

- User Authentication
- SOS Alert
- Current Location
- Emergency Contacts
- AI Safety Chatbot
- Nearby Police Stations
- Feedback System
- Responsive Design

## Tech Stack

### Frontend
- React.js
- Vite
- CSS
- Axios

### Backend
- Node.js
- Express.js
- MySQL

## Installation

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm run dev
```
